# Chezmoi plugin

The chezmoi plugin defines a number of useful aliases for you. [Consult the complete list](./chezmoi.plugin.sh)

## List of aliases

| Alias | Command                |
| ----- | ---------------------- |
| cz    | 'chezmoi'              |
| cza   | 'chezmoi add'          |
| czcd  | 'chezmoi cd'           |
| cze   | 'chezmoi edit'         |
| czea  | 'chezmoi edit --apply' |
| czra  | 'chezmoi re-add'       |
